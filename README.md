<h1 align="left">🎸 Concerts</h1>

###

<p align="left">Одностраничный сайт, посвящённый музыкальным концертам и фестивалям. Проект создан для демонстрации событий, артистов и продажи билетов в современном формате.</p>

###

<h2 align="left">🧑‍💻 Технологии сайта</h2>

###

- HTML5 — семантическая разметка

- SCSS — модульные стили

- JavaScript — интерактивные элементы

- API поддержка

- Vite — быстрая сборка и dev-сервер

- GitHub Pages — деплой

###

<h2 align="left">📁 Структура проекта</h2>

###

```
concerts/
├── src/
│   ├── index.html        # Главная страница
│   ├── styles/           # SCSS стили
│   └── js/               # Скрипты
├── .github/workflows/    # CI/CD конфигурация
├── vite.config.js        # Конфигурация Vite
├── package.json          # Скрипты и зависимости
└── README.md             # Описание проекта
```
